Making a snake game in <canvas>

The canvas element

  • A fixed-size HTML element
  • Defines an area of the page that can be drawn on
  • Can be transparent (used as an overlay)
  • It's a blank canvas

Getting the context

The canvas isn't used directly - you draw on an object called its context.

The context is obtained by calling getContext('2d') on the canvas element. (There may be a 3d canvas element in the future, but for now all we have is 2d).

Making a game loop

The game loop is a function that is called at a set interval, and updates the game.

  • Update the game state
  • Update the display to reflect the new state

Drawing a snake

Part 1

The snake is drawn as a collection of squares. It's represented as an array of co-ordinates, with the head at the beginning.

Each frame adds a new position to the front of the snake and removes the last one.

Each time round the game loop:

  • snake.advance() is called to move the snake
  • snake.draw() is called to draw the snake on the canvas.
  • draw() calls drawSection which draws each square

Adding an apple

The apple is drawn as a circle. To create a circle in canvas, call on the context:

  • beginPath()
  • arc(x, y, radius, startAngle, endAngle, antiClockwise?)
  • fill()
